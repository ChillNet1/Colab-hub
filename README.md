# Colab-hub (Video Conferencing & Collaboration)

Web application for video conferencing, screen sharing, chat, and collaborative whiteboard.

## Features
- Video conferencing (camera / microphone control)
- Screen sharing
- Real-time chat
- Collaborative whiteboard
- Lobby system for joining sessions

## Tech Stack
- Vite
- React + TypeScript
- Node.js
- WebRTC (or signaling server if used)

## Setup

```bash
npm install
npm run dev
